자동화 흐름

5단계가 모두 자동으로 실행됩니다. 담당자는 알림만 확인하면 됩니다.

01

📧 주문서 이메일 수신

거래처에서 주문서(PDF/이미지)가 전용 이메일로 도착합니다. 양식이 달라도 상관없습니다.

전용 이메일 계정
02

🤖 AI 자동 파싱

Claude AI가 주문서를 읽고 품목·수량·납기·거래처를 자동으로 추출합니다. 거래처별 다른 양식도 모두 인식합니다.

Claude AI
03

💾 SKE ERP 수주 자동 등록

파싱된 데이터가 사내 ERP 수주관리 화면에 자동으로 등록됩니다. 중복 주문은 자동 필터링됩니다.

SKE ERP (Firebase)
04

📡 이카운트 연동

이카운트 ERP에도 수주 전표가 자동 생성됩니다. 이중 입력이 완전히 사라집니다.

이카운트 Open API
05

📲 텔레그램 즉시 알림

담당자 휴대폰으로 수주 내용이 즉시 전송됩니다. 어디서든 확인 가능합니다.
